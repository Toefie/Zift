Favicon Guide for Zift

Favicons are the small icons that appear in browser tabs, bookmarks, and other areas to represent your website. Proper favicon implementation enhances brand recognition and professionalism.

Important: The placeholder favicon files need to be replaced with actual icon files. Text files won't display as icons in browsers.

Required Favicon Files

For optimal cross-browser and device support, you should replace these placeholder files with actual icon files:

  • favicon.ico - 16x16 and 32x32 (multi-size ICO file)
  • favicon-16x16.png - 16x16 PNG file
  • favicon-32x32.png - 32x32 PNG file
  • apple-touch-icon.png - 180x180 PNG file for Apple devices

How to Generate Proper Favicon Files

You can use online tools to generate all required favicon files from your logo:

  1. Visit a favicon generator website like favicon.io or realfavicongenerator.net
  2. Upload your logo or create a custom icon
  3. Download the generated favicon package
  4. Replace the placeholder files with the generated files

HTML Implementation

The following code is already added to your website's HTML head section:

<!-- Favicon -->
<link rel="icon" href="favicon.ico" type="image/x-icon">
<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">
<link rel="apple-touch-icon" href="apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="favicon-16x16.png">
Tip: For the best favicon display, ensure your icon has a simple design that's recognizable even at small sizes. Typically, using your logo or a simplified version of it works well.
